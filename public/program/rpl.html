TKJ RPL DKV

Profil Program

Program RPL fokus pada siklus pengembangan perangkat lunak lengkap — dari analisis kebutuhan, desain, coding, pengujian, hingga deployment. Siswa belajar dengan proyek nyata dan kolaborasi tim seperti di industri startup dan korporasi IT.

Kompetensi Utama

  • Pemrograman web frontend dan backend
  • Pengembangan aplikasi mobile dasar
  • Desain dan implementasi basis data
  • UI/UX dan metodologi Agile/Scrum

Prospek Karier

Lulusan RPL siap menjadi junior developer, web developer, mobile developer, atau QA engineer. Banyak alumni langsung diterima magang dan kerja di mitra industri sekolah.
